# Converget-TCR
A respository of examples of computational techniques for analyzing T cell receptors (TCRs) in the immune system

## TCR Informatics in Metastatic Renal Cancer
Evaluation of patientsâ€™ T and B cell repertoires is a potentially powerful avenue for diagnosis and therapeutic-response monitoring of immunotherapies. New methodologies for identifying and tracking cancer-specific CDR3 regions from the estimated 10^15-10^25 total diversity of TCR CDR3 sequences could allow for early prediction of cancer states and patient responses. However, development of these diagnostic tools is hampered by an unmet need for data-analytics tools capable of modeling small sample sizes. While LLMs gain a greater ubiquitous hold in the small molecule informatics realms, there is still a strong need to leverage statistical methods capable of handling high-feature data with low sample size. This work seeks to featurize and cluster TCRs by sequence similarity using motif analysis.
